About the app: This web application sets out to provide users with a simple viewing of particular top stocks in the market. Users should be able to view detailed information about the stock, and see prices changes every so often, as well as save stocks they are interested in, to a wishlist. Users also will have access to a live news feed of stories coming from Benzinga. This outlet provides news stories on financial/trading topics.
Please note that this is a demo webapp utilizing free apis. Prices will not update as frequent as normal brokerage sites, and refreshing/loading pages too quickly can cause the api usage limit to be exceeded, resulting in empty data.
